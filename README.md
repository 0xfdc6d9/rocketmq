### RocketMQ是什么？
淘宝Metaq从3.0版本开始，改名为RocketMQ。RocketMQ是一款分布式、队列模型的消息中间件，具有以下特点：

* 能够保证严格的消息顺序
* 提供丰富的消息拉取模式
* 高效的订阅者水平扩展能力
* 实时的消息订阅机制
* 亿级消息堆积能力
* 极少的依赖，非常适合外部用户使用
